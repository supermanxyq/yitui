# 图片资源说明

请将以下图片资源放置在此目录中，以确保网站正常显示：

## 必要图片

- `logo.png` - 网站 logo (建议尺寸: 150 x 40 px)
- `hero-bg.jpg` - 首页大横幅背景图 (建议尺寸: 1920 x 800 px)
- `icon1.png` ~ `icon8.png` - 核心业务能力图标 (建议尺寸: 60 x 60 px)
- `app-mockup.png` - APP 界面展示图 (建议尺寸: 300 x 600 px)
- `team1.jpg`, `team2.jpg` - 团队成员照片 (建议尺寸: 400 x 500 px)
- `case1.jpg` ~ `case4.jpg` - 案例展示图片 (建议尺寸: 400 x 300 px)
- `tech-diagram.png` - 技术架构图 (建议尺寸: 800 x 500 px)
- `qrcode.png` - 微信公众号二维码 (建议尺寸: 200 x 200 px)

## 图片优化建议

为了提高网站加载速度，建议：

1. 优化所有图片，控制文件大小
2. 使用适当的图片格式：
   - 照片类图片使用 JPG 格式
   - 图标和简单图形优先使用 PNG 或 SVG 格式
3. 背景图片考虑使用 WebP 格式，并提供 JPG 备选
4. 图标建议使用 SVG 矢量格式，以保证在各种尺寸下的清晰度

## 临时图片资源

开发测试期间，可以使用以下在线服务生成临时图片：

- [Placeholder.com](https://placeholder.com/) - 生成纯色占位图
- [Lorem Picsum](https://picsum.photos/) - 生成随机美观的照片
- [Undraw](https://undraw.co/) - 提供免费的 SVG 插图
